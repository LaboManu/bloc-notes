J-Painter - Online image editor.

www.izhuk.com/painter

J-Painter is an image editor which works within a browser.
The images can be saved to and loaded from the server.

The developer's guide

The developer's guide can be found here.

Examples

The examples show how to display the editor and save images on your server.
If you have a local web server installed on your desktop you can run the examples now.
For that open this page through your local web server and follow the links below.

PHP Example
ASP.NET Example
ASP Classic Example
JSP Example

If some of the examples are not interesting for you, you can remove their directories harmlessly,
because each example is self-contained and is not related to the other ones.

Directory structure

applet	 - contains the editor itself and its resources
doc	 - the developer's guide
php	 - PHP example
aspx	 - ASP.NET example
asp	 - ASP Classic example
jsp	 - JSP example
cliparts - example images of cliparts and JavaScript for putting them on the Draw Area

Using options

J-Painter can be used either as service or product.

Service

Service is provided for free and can be used for testing, development, or unimportant activities. The service assumes that computers of your users can connect to the host izhuk.com when they are drawing.

Product

You can purchase J-Painter as product and install its server-side components on your server or servers of your customers. It is necessary when your users can't connect to izhuk.com, for example in Intranet environment, or you think this connection is undesirable, or you need independancy for any other reason.
